# The Maze That Remembered

------------------------------------------------------------------------

# Purpose

This document defines the quality gates that every chapter must pass
before being considered complete.

Writing is creative.

Editing is engineering.

A chapter is not "finished."

It has simply passed every required quality gate.

------------------------------------------------------------------------

# Quality Philosophy

Every revision must improve at least one measurable property of the
manuscript.

Revise because a criterion failed---not because of vague intuition.

------------------------------------------------------------------------

# Five Review Layers

Every chapter must pass:

1.  Story
2.  Character
3.  Science
4.  Language
5.  Reader Experience

Failure in any layer returns the chapter to revision.

------------------------------------------------------------------------

# Story Review

Questions:

-   Does the chapter begin with curiosity?
-   Does something meaningful change?
-   Does the ending naturally create the next question?
-   Would someone continue reading even without interest in NeuroAI?

Failure if exposition replaces narrative.

------------------------------------------------------------------------

# Character Review

Verify:

-   Elias grows naturally.
-   Mira teaches through questions rather than lectures.
-   Dialogue sounds spoken rather than written.
-   Silence is used intentionally.

Failure if characters become educational mouthpieces.

------------------------------------------------------------------------

# Scientific Review

Consult:

SCIENTIFIC_DEPENDENCY_MAP.md

Verify:

-   exactly one major concept is introduced
-   prerequisites are satisfied
-   analogies remain scientifically faithful
-   no future concepts appear prematurely

------------------------------------------------------------------------

# Language Review

Ask:

-   Can this sentence be shorter?
-   Can a stronger verb replace adjectives?
-   Can implication replace explanation?
-   Can silence replace dialogue?

Remove unnecessary words before adding new ones.

------------------------------------------------------------------------

# Reader Review

Verify:

-   the reader's incoming mental model
-   the transformed mental model
-   one memorable image
-   one memorable question
-   one lasting conceptual shift

Learning should feel like discovery.

Never memorization.

------------------------------------------------------------------------

# Symbol Review

Consult SYMBOL_BIBLE.md.

Every recurring symbol must deepen.

Never repeat unchanged.

------------------------------------------------------------------------

# Continuity Review

Consult CONTINUITY_GUIDE.md.

Verify:

-   character continuity
-   scientific continuity
-   symbolic continuity
-   emotional continuity
-   callbacks
-   foreshadowing

------------------------------------------------------------------------

# Cognitive Load

One chapter.

One transformation.

Never introduce multiple major conceptual revolutions simultaneously.

------------------------------------------------------------------------

# Rewrite Order

Always:

1.  Delete
2.  Reorganize
3.  Rewrite
4.  Polish

Never polish structural problems.

------------------------------------------------------------------------

# Chapter Acceptance Checklist

Story ............ PASS

Character ........ PASS

Science .......... PASS

Language ......... PASS

Reader ........... PASS

Continuity ....... PASS

Symbols .......... PASS

One Transformation PASS

Only then may the chapter be marked:

LOCKED

------------------------------------------------------------------------

# Manuscript Review

After completing the novel verify:

-   every chapter is necessary
-   every concept returns
-   every symbol matures
-   every callback rewards attention
-   the ending feels inevitable

If not,

unlock the manuscript and revise.

------------------------------------------------------------------------

# Definition of Done

The manuscript is complete when:

-   no chapter can be removed
-   no concept can be moved
-   no sentence can be shortened without losing meaning
-   every scientific intuition remains faithful
-   readers finish asking better questions than when they began

------------------------------------------------------------------------

# Final Principle

Quality is measured not by how much the manuscript contains,

but by how little can still be removed.
