# README.md

# The Maze That Remembered

*A literary scientific novella introducing NeuroAI through story.*

Version: 1.1 (Pre-Manuscript)

------------------------------------------------------------------------

# What is this repository?

This repository contains everything required to produce **The Maze That
Remembered**.

It is **not** the manuscript.

It is the production system behind the manuscript.

Think of this repository as the operating manual for writing the book.

The manuscript is only one component.

Everything else exists to keep the manuscript coherent, scientifically
accurate, emotionally engaging, and faithful to its original vision.

------------------------------------------------------------------------

# Project Mission

Create the book we wished existed before learning computational
neuroscience.

The goal is not to simplify NeuroAI.

The goal is to make its ideas *discoverable*.

Readers should feel that they arrived at each concept naturally through
experience rather than through exposition.

The book should function simultaneously as:

-   literary novella
-   cognitive apprenticeship
-   conceptual introduction to NeuroAI
-   scientific coming-of-age story

without ever becoming a textbook.

------------------------------------------------------------------------

# Length Philosophy

The manuscript is intentionally concise.

Every page must earn its existence.

The book should be readable over two or three evenings, yet remain
memorable for years.

**Target length**

-   25,000--35,000 words
-   Approximately 100--120 printed pages

Brevity is a feature, not a limitation.

------------------------------------------------------------------------

# Guiding Principle

> **Every scientific concept must be earned before it is named.**

Readers should first experience a problem.

Then discover the need for a concept.

Only afterwards encounter its terminology.

------------------------------------------------------------------------

# Core Question

Everything in this project ultimately serves one question:

> **How do we build better models of reality?**

The story is one answer.

NeuroAI is another.

The reader discovers that they are the same journey.

------------------------------------------------------------------------

# Repository Philosophy

Planning is frozen.

Writing is iterative.

Revision is expected.

The architecture remains stable while the manuscript improves.

# Definition of Success

This project succeeds if readers close the book eager---not
intimidated---to read real NeuroAI papers, textbooks, and code.

The novella should change how they think, not merely what they know.

# Final Reminder

Whenever uncertainty arises, return to the central promise:

> **Create experiences from which understanding becomes unavoidable.**
