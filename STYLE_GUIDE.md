# STYLE_GUIDE.md

# The Maze That Remembered

Version: 2.0

Status: Frozen Before Draft One

------------------------------------------------------------------------

# Purpose

This document defines the literary voice of the manuscript.

It is a specification for style, not a collection of writing tips.

The goal is consistency across every page.

------------------------------------------------------------------------

# The Golden Rule

> Write one level quieter than your instinct.

Prefer restraint over performance.

Readers should notice the ideas, not the prose.

------------------------------------------------------------------------

# Desired Reading Experience

The novel should feel like:

-   entering an old library
-   walking through a garden after rain
-   discovering rather than being instructed
-   thinking alongside the characters

Never rushed.

Always inviting.

------------------------------------------------------------------------

# Narrative Voice

The narrator is calm, observant, and nearly invisible.

The narrator never lectures.

The narrator trusts the reader.

------------------------------------------------------------------------

# Sentence Philosophy

Every sentence should move either:

-   the story,
-   the reader's understanding,
-   or the emotional state.

If it does none of these, remove it.

------------------------------------------------------------------------

# Dialogue

Dialogue reveals thought.

It never exists solely to transfer information.

Characters speak naturally.

No speeches.

No TED Talks.

------------------------------------------------------------------------

# Scientific Language

Scientific terms appear only after readers need them.

Definitions satisfy curiosity.

They never create it.

------------------------------------------------------------------------

# Description

Favor concrete observations over abstract commentary.

Show:

-   stone
-   paper
-   rain
-   footsteps
-   light

before discussing ideas.

Readers think through physical experience.

------------------------------------------------------------------------

# Metaphors

Use few.

Each metaphor must:

-   clarify
-   remain scientifically faithful
-   retire before it becomes misleading

------------------------------------------------------------------------

# Emotional Writing

Show emotion through:

-   attention
-   hesitation
-   silence
-   behavior

Avoid explaining feelings directly.

------------------------------------------------------------------------

# Pacing

Alternate naturally between:

Observation

↓

Conversation

↓

Experiment

↓

Reflection

Allow the manuscript to breathe.

------------------------------------------------------------------------

# Vocabulary

Prefer:

observe

notice

perhaps

wonder

still

quietly

Avoid whenever possible:

obviously

clearly

actually

simply

very

really

------------------------------------------------------------------------

# Literary North Star

Readers should finish chapters feeling:

more curious,

more attentive,

and more capable of thinking.

Never merely impressed.

------------------------------------------------------------------------

# Final Test

If every scientific term disappeared,

would someone still enjoy reading the scene?

If not,

rewrite until the scene succeeds as literature first.

------------------------------------------------------------------------

# One Sentence

Write with enough restraint that readers mistake discovery for their own
thinking.
