# Purpose

This document defines the scientific review process for the manuscript.

The novella is fiction.

Its scientific ideas are not.

The objective is to ensure every intuition the reader develops remains
compatible with modern computational neuroscience, machine learning, and
NeuroAI.

------------------------------------------------------------------------

# Guiding Principle

The manuscript teaches durable intuition rather than exhaustive detail.

Scientific review evaluates:

-   correctness
-   faithfulness
-   pedagogical value
-   future compatibility

------------------------------------------------------------------------

# Reviewer Mindset

Review as if mentoring an exceptionally curious first-year graduate
student.

Ask:

"Will this intuition remain useful when the reader later studies the
mathematics?"

If unlearning would be required,

the explanation fails.

------------------------------------------------------------------------

# Core Domains

Review the fifteen major concepts and the supporting Hypothesis bridge
using their exact names and definitions in
SCIENTIFIC_DEPENDENCY_MAP.md.

No adjacent neuroscience, machine-learning, or NeuroAI topic is
authorized merely because it would normally accompany one of those
concepts.

------------------------------------------------------------------------

# Source Boundary

SCIENTIFIC_DEPENDENCY_MAP.md is the manuscript's complete scientific
canon.

External sources may be consulted only to verify the accuracy of a
listed concept. They may not supply new terminology, mechanisms,
examples, claims, or scientific domains to the manuscript.

If the repository does not contain enough information to write a claim
honestly, stop and report the gap. Never fill it by importing a new
concept.

------------------------------------------------------------------------

# Concept Review

For every scientific concept evaluate:

-   Is it technically correct?
-   Is it introduced after its prerequisites?
-   Does it answer a question the reader already possesses?
-   Does it avoid unnecessary jargon?
-   Could it produce a misconception?

------------------------------------------------------------------------

# Analogy Review

Each analogy must include:

Purpose

Strengths

Limitations

Retirement point

Retire analogies before they become scientifically misleading.

------------------------------------------------------------------------

# Common Failure Modes

Reject explanations that encourage:

-   anthropomorphism
-   localization of distributed representations
-   static views of computation
-   behavior-only explanations
-   transparency equals understanding

------------------------------------------------------------------------

# Dependency Validation

Consult:

SCIENTIFIC_DEPENDENCY_MAP.md

No concept may violate the dependency graph.

------------------------------------------------------------------------

# Reader Validation

Consider four readers:

1.  General reader
2.  Undergraduate student
3.  Beginning PhD student
4.  Domain expert

The manuscript succeeds when:

-   beginners understand
-   experts respect the simplification

------------------------------------------------------------------------

# Terminology

Scientific terms should be:

-   introduced once
-   used consistently
-   reinforced through application

Never overload terminology.

------------------------------------------------------------------------

# Review Checklist

For each chapter verify:

-   scientific accuracy
-   dependency order
-   analogy fidelity
-   conceptual clarity
-   future compatibility
-   reader intuition
-   misconceptions avoided

------------------------------------------------------------------------

# Completion Boundary

Chapter-level scientific review and the final internal scientific pass
are required for **Manuscript Complete**.

Feedback from the four reader groups is required for **Publication
Ready** and is coordinated by the user after the manuscript is complete.
The drafting agent must not fabricate external reviewers or their
responses.

------------------------------------------------------------------------

# Definition of Scientific Success

The manuscript succeeds scientifically when:

-   every simplification is honest
-   every intuition survives later formal study
-   experts disagree only about emphasis, not correctness

------------------------------------------------------------------------

# Final Principle

The novella is a bridge.

Its purpose is not to replace textbooks.

Its purpose is to prepare readers to love them.
