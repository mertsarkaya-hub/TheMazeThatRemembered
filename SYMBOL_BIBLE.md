# SYMBOL_BIBLE.md

# The Maze That Remembered

Version: 2.0

Status: Frozen Before Draft One

------------------------------------------------------------------------

# Purpose

This document defines every recurring symbol in the manuscript and how
its meaning evolves.

Symbols are never decorative.

They accumulate meaning as the reader changes.

------------------------------------------------------------------------

# Symbol Philosophy

Readers should not think:

"This symbolizes..."

They should think:

"This feels different now."

Meaning grows through recurrence.

Never through explanation.

------------------------------------------------------------------------

# Growth Model

Every recurring symbol follows four stages:

1.  Object
2.  Tool
3.  Concept
4.  Mirror

No symbol skips stages.

------------------------------------------------------------------------

# Rain

Never symbolizes sadness.

It teaches attention.

Readers learn to notice environments, not just events.

------------------------------------------------------------------------

# Light

Represents changing perspective.

As Elias learns, different things become visible.

------------------------------------------------------------------------

# Stone

Continuity.

Questions outlive individuals.

------------------------------------------------------------------------

# Wood

Construction.

Ideas are built.

------------------------------------------------------------------------

# Glass

Visibility is not understanding.

Transparency alone never explains a system.

------------------------------------------------------------------------

# Silence

Silence is an active symbol.

Different silences carry different meanings:

-   uncertainty
-   observation
-   reflection
-   satisfaction

------------------------------------------------------------------------

# Symbol Density

No scene should actively develop more than two symbols.

Too many symbols weaken each other.

------------------------------------------------------------------------

# Recurrence Rule

Every return must deepen meaning.

Never repeat a symbol merely to remind readers it exists.

------------------------------------------------------------------------

# Final Return

Every major symbol introduced early should return near the ending.

Not as nostalgia.

As conceptual completion.

------------------------------------------------------------------------

# Final Test

Ask:

What new layer of meaning has the reader earned since the last
appearance?

If the answer is "none,"

remove the recurrence.

------------------------------------------------------------------------

# One Sentence

The symbols change because the reader changes.
