# PARKING_LOT.md

# The Maze That Remembered

Version: 2.0

Status: Living Document

------------------------------------------------------------------------

# Purpose

This document protects the manuscript from scope creep.

Every exciting idea is preserved.

Not every exciting idea belongs in Draft One.

The Parking Lot is a vault, not a trash bin.

------------------------------------------------------------------------

# Core Principle

Every new idea must answer two questions:

1.  Is it a good idea?
2.  Is this the right time?

Only ideas with two "yes" answers enter the manuscript.

Everything else comes here.

------------------------------------------------------------------------

# Temptation Log

Whenever you think:

"I should explain this now..."

Write it here.

Whenever you think:

"This reminds me of another concept..."

Write it here.

Then continue drafting.

Momentum is sacred.

------------------------------------------------------------------------

# Graduation Criteria

An idea may leave the Parking Lot only if it:

-   strengthens the central promise
-   preserves one transformation per chapter
-   introduces no broken dependencies
-   improves the reader's journey more than it increases complexity

Otherwise it stays.

------------------------------------------------------------------------

# Cost of Every New Idea

Every addition increases:

-   complexity
-   continuity burden
-   revision effort
-   cognitive load
-   scientific review

Nothing is free.

------------------------------------------------------------------------

# End-of-Project Review

After Draft One classify every entry:

-   Adopt
-   Reserve for future edition
-   Develop into companion material
-   Discard

No idea should be forgotten.

------------------------------------------------------------------------

# Final Principle

Restraint is respect for the reader.

A great manuscript contains only the ideas that belong together.

------------------------------------------------------------------------

# One Sentence

Every unwritten idea deserves to be remembered.

Not every remembered idea deserves to be written.
